康养城市建设与优化配置建模
摘要
康养城市建设不仅是国家层面的战略部署，也是地方发展的实际需求。本研究旨在
解决如何在有限的资源条件下，优化康养设施布局，提升康养服务质量的问题，以推动
智慧养老与康养产业的深度融合。
针对问题1，本研究以北京市16 个区为对象，围绕康养资源分布的现状分析与优化
需求展开。通过收集2023 年北京市各区康养资源及居民健康状况数据，进行数据处理
与可视化分析，明确了各区康养资源和居民健康指标的空间分布特征。结果显示，中心
城区康养资源富集，远郊区资源匮乏，且居民健康指标存在显著区域差异；进一步构建
康养资源指数（Ri）和居民健康指数（Hi）模型，采用熵值法确定权重，整合多维度指
标。经相关性分析发现，剔除东城区、西城区后，其余14 区康养资源指数与居民健康
指数呈正相关，相关系数0.6834，但整体方差达0.7075，表明资源分配不均衡。引入资
源匹配度（PRi）评估发现，超半数区域存在人均资源不足问题；为优化资源布局，构
建以最小化康养资源指数方差和平均可达时间为目标的优化模型，采用SLSQP 算法求
解。结果表明，优化后需减少东城区、西城区、石景山区的康养资源量，增加朝阳区、
丰台区等区域的资源配置，以提升服务公平性与可达性，为康养城市建设提供科学规划
依据。
针对问题2，本研究构建了一套完整的评价体系与优化方法，以评估和提升北京市
康养城市发展水平。具体而言，首先从多个维度选取代表性定量指标，运用熵权法与
TOPSIS 方法构建多维综合评价模型，对北京市各区的康养城市发展水平进行量化评估。
结果表明，不同区发展水平差异较大，存在明显的两极分化。在此基础上，创新性地引
入强化学习方法，定量计算并提出各区在各维度上的具体发展调整建议，形成一套可量
化、可操作的发展优化方案。
针对问题3，本研究构建康养资源多目标优化配置模型，以北京市16 区为对象，
设定覆盖率、效率最大化与成本最小化目标函数，结合服务半径、预算等约束条件，采
用粒子群优化（PSO）、差分进化算法（DE）、NSGAIII 算法求解。结果表明，海淀
区、房山区等区域需针对性增加医疗、养老等设施数量（如房山区医疗设施需增3111
家），部分核心城区设施配置可维持合理水平。研究提出区域差异化政策、多元资金投
入及智慧康养技术等优化策略，为康养资源高效配置提供科学方案。
关键词：康养城市建设；北京市；康养资源；居民健康指数；SLSQP 法；TOPSIS 法；

1. 问题重述
1.1 问题背景
随着全球老龄化进程加速，康养城市建设已成为应对人口结构变化、提升居民生活
质量的重要战略方向。2025 年全国两会期间，多位代表委员围绕康养产业转型、智慧
养老及医养融合等议题提出关键建议，如盘活“空置房” 用于康养产业、发展数字家庭
技术等。安徽省人民政府发布《关于支持康养产业高质量发展的意见》，强调康养产业
与医疗、旅游、文化的深度融合；地方实践中，广德市推进“康养名城” 战略，通过优
化设施布局、提升智慧养老水平等打造康养目的地，吉林省亦将旅居康养纳入地方文旅
产业发展规划。此类政策与实践表明，康养城市建设既是国家战略部署，亦是地方经济
社会发展的现实需求。
然而，当前康养城市建设面临多重挑战。康养资源（如医疗设施、养老机构、文化
设施等）在区域间分布不均衡，智慧养老技术应用滞后，医养融合深度不足。在资源总
量有限的约束下，如何通过科学规划优化康养设施布局、提升服务可达性与公平性，成
为亟待解决的关键问题。
1.2 问题提出
针对康养城市建设问题，要求我们通过参考并收集相关数据，比如：2024 年中国康
养城市100 强名单，国家统计局数据（老年人口数据、居民收入与消费支出、生态环境、
医疗水平）、人力资源和社会保障部（养老保险基金支出）以及一些国内外公开的数据，
通过数学建模（或算法）方法，解决以下问题：
（1）问题一是康养资源分布的现状分析与优化需求问题。要求我们收集或使用某
城市不同区域模拟数据，包括康养资源数据（医疗设施、养老机构、公园绿地、文化设
施的数量与分布密度）以及居民健康状况数据（平均寿命、慢性病发病率、健康满意度
等），进而开展研究。评估所选城市的康养资源分布的空间特征，判断康养资源的富集
区和匮乏区，通过分析康养资源分布与居民健康指标的相关性，判断当前康养资源分布
的合理性。在此基础上，针对康养资源分布不均衡的问题，提出优化布局的初步思路，
重点提升康养服务的公平性与可达性。
（2）问题二是康养城市综合评价模型的构建与应用问题。康养城市建设已成为提
升城市竞争力与居民生活质量的重要方向，需要科学的评价体系来衡量其发展水平。现
需构建一个适用于康养城市的综合评价模型，具体表现为，首先，需从康养资源丰富度
（如医疗设施数量、养老机构床位密度等）、居民健康状况（包括平均寿命、慢性病发
病率等）、环境质量（如空气质量指数、绿地覆盖率等）、经济发展水平（人均GDP、
康养产业占比等）等多个核心维度，合理选取并定义具体的评价指标；其次，利用该模
型对目标城市的康养水平进行量化计算，通过数据处理与分析，明确其在各维度上的优
势与不足；最后，基于分析结果，提出具有针对性的改进建议，包括资源优化配置方案、
环境治理措施、产业发展策略等，为康养城市的科学建设提供可操作的决策依据

（3）问题三是康养资源优化配置模型的建立与实施策略问题。康养城市建设的核
心在于高效配置康养资源，以提升康养服务的覆盖率和资源利用效率，同时在有限的资
源条件下最小化成本。当前康养城市建设面临诸多挑战，如康养资源分布不均衡、智慧
养老技术应用不足、医养融合深度不够等。这些问题导致部分区域康养资源过剩，而另
一些区域则资源匮乏，影响了康养服务的公平性和可达性。因此，需要建立一个科学的
康养资源优化配置模型，通过多目标优化方法，合理规划康养设施的选址、数量和服务
范围，以实现资源的最优配置。具体问题为如何在有限的资源条件下，最大化康养服务
的覆盖率和资源利用效率，同时最小化成本。

2. 问题分析
问题一：针对问题1 的要求，其求解可分为以下四步，首先选择某一典型城市为研
究对象，收集并整理题目所需要的相关数据，并利用Python 编程语言和Geopandas 库进
行数据处理和可视化分析，明确该城市各区康养资源和居民健康状况各指标的空间分布
特征并进行初步的定性分析，判断康养资源分布的合理性；随后进行定量分析，综合各
个独立指标，建立康养资源指数和居民健康指数模型，用以反映康养资源服务压力和居
民健康水平；之后，构建康养资源合理性定量评价模型，使用采用Pearson 相关性分析
法，明确康养资源指数与居民健康指数的关系，分析资源配置不合理的情况，为了进一
步探究资源分配的均衡性，基于康养资源指数，探究其离散程度；最后构建资源分配优
化模型，分析该城市各区资源匹配度，确定目标函数和约束条件，并确定合适的优化方
法进行计算，确定资源分配方案，以解决资源分布不均衡的问题，提高康养服务的公平
性和可达性。
问题二：针对问题2 的要求，核心任务是建立一个合理、全面的综合评价模型，量
化衡量城市康养发展水平，分析当前城市康养水平的优势与不足，并为城市康养规划提
供决策支持。对于康养城市发展水平的评价，需从多维度构建综合评价体系。一方面，
要考量康养资源的丰富度与配置效率，涵盖医疗设施的可达性、养老机构的服务能力、
公园绿地的空间分布以及文化设施的覆盖密度等，确保资源在城乡、区域间均衡布局；
另一方面，需聚焦居民健康状况，包括平均寿命、慢性病发病率等。同时，环境质量是
关键基础，涉及空气质量、绿地面积等，直接影响居民的生活舒适度与健康风险。经济
发展水平则为康养服务提供支撑，涉及人均GDP、地区生产总值等，确保产业可持续发
展。从而形成涵盖资源、健康、环境、经济等多维度的立体评价框架。因此，针对问题
二，选取各个维度下具有代表性的定量指标，我们设计了基于熵权和TOPSIS 方法的
多维综合评价模型来评价北京市康养城市的发展水平，根据评价结果进行聚类对多个区
进行发展水平等级划分，并通过多种可视化分析北京市不同地区发展水平及分布差异，
最后创新性的应用了强化学习方法进行提出改进意见，该方法定量的指出了每个区未来
应该做出的发展调整从而提高发展水平。
问题三：针对问题3 的要求，本质为多目标优化问题。首先，应平衡研究城市各区
最大化服务覆盖率、资源利用效率和最小化成本这三个目标，考虑设施选址等空间约束。
构建模型时，确定决策变量，设好最大化覆盖率、效率和最小化成本的目标函数，服务
半径、预算等约束条件。在多目标优化方法选择上，拟采用粒子群优化（PSO）、差分
进化算法（DE）、第三代非支配排序遗传算法（NSGAIII）三种前沿优化算法对结果进
行分析总结。最后依据三种算法模型结果与问题一结果进行对比分析、合理的提出具体
的实施策略，包括政策建议、资金投入方向、技术创新措施等。

3. 模型假设与符号说明
3.1 模型基本假设
为了简化模型并使其更贴近实际情况，本研究做出以下假设：
（1）资源总量固定。假设北京市康养资源总量在优化过程中保持不变，即资源的
增加或减少仅在不同区域之间进行调配；
（2）区域独立性。假设各区域之间的康养资源分配相互独立，不考虑跨区域的资
源共享或协同效应。
（3）数据准确性。假设收集的康养资源和居民健康状况数据是准确且可靠的，能
够真实反映各区域的实际情况。
（4）人口稳定性。假设在研究期间，各区域的人口数量保持相对稳定，不考虑大
规模的人口流动对康养资源需求的影响。

4. 模型建立与求解
4.1 问题一模型建立与求解
4.1.1 数据与分布特征
（1）研究对象选择与数据收集
康养城市的建设离不开对现有康养资源的全面评估。北京作为中国的首都，人口规
模庞大且老龄化程度高，截至2023 年，60 岁及以上常住人口占比达20.2%，65 岁及以
上常住人口占比14.9%，康养资源需求巨大。同时，北京市各区在经济发展水平、人口
密度、土地利用等方面存在显著差异，这种差异导致各区对康养资源的需求和供给能力
不同，研究其康养资源分布情况，能够更好地了解和解决区域间的康养资源不均衡问题，
实现资源的优化配置。此外，北京市政府高度重视康养事业的发展，出台了一系列政策
文件，为研究提供了明确的方向和依据。同时，北京的数据资源丰富且相对容易获取，
能够为研究提供坚实的基础，确保研究结果的准确性和可靠性[1,2]。
因此，本团队针对此问题，选择北京市各区作为康养资源研究对象，不仅具有重要
的现实意义，还具有典型性和示范性，能够为其他城市提供借鉴和参考，推动全国康养
事业的发展。本次收集并整理了2023 年北京市东城区、西城区、朝阳区、丰台区、石
景山区、海淀区、门头沟区、房山区、通州区、顺义区、昌平区、大兴区、怀柔区、平
谷区、密云区及延庆区等16 个区的康养资源分布情况（医疗设施、养老机构、公园绿
地、文化设施等的数量和密度），以及居民健康状况数据（平均寿命，慢性病发病率，
慢性病过早死亡率、健康满意度等）。部分数据如下表所示。

4.2.1 问题求解思路

科学指导。

4.2.5 基于强化学习的改进方案
为提出定量化改进意见，进一步收集了2018-2022 年康养指标数据，构建包含多维
动态状态空间的马尔可夫决策过程，借助PPO 强化学习算法[5,6]；以2023 年数据作为初
始状态，通过模拟未来的规划，将过去六年历史数据隐含的发展规律融入状态空间构建；

Proximal Policy Optimization（PPO）算法是强化学习中一种高效的策略梯度方法，
PPO 通过引入剪切概率比（Clipped Surrogate Objective）替代传统策略梯度，显著提高
了训练的稳定性与收敛速度。PPO 属于基于策略的近端优化算法，其目标是通过迭代优
化，学习一个最优策略，以在给定状态下输出最优动作，最大化期望累计奖励。
在本文中，PPO 被应用于康养城市发展水平优化问题，通过调整各区域的医疗资源、
养老设施、绿地分布等多维康养资源配置（即动作），以提升基于TOPSIS 模型计算的
康养发展水平得分（即奖励），实现康养城市长期健康可持续发展目标。
策略（Policy）：从状态到动作的映射关系，由神经网络参数化。
（2）环境与状态空间设计
在康养城市优化场景中，PPO 算法将康养发展规划建模为马尔可夫决策过程（MDP）：
状态（State）：城市各康养指标的当前状态（如医疗设施密度、公园绿地密度等）。
动作（Action）：对各指标的调整策略（如增加医疗设施）。
奖励（Reward）：基于康养综合得分的反馈信号。
（3）网络结构
包括策略网络和价值网络。

标），再通过两层两层全连接层，每层128 个神经元，激活函数为ReLU；输出为动作
均值向量（对应各指标的调整比例）和对数标准差向量。
价值网络：输入层为标准化状态向量（包含医疗设施密度、养老机构密度等康养指

价值估计。

4.3.5 NSGAIII 改进模型建立及求解
NSGA III（基于参考点的非支配遗传算法）是一种专门针对多目标优化问题的算法，
NSGAIII 是NSGAII 的改进版本，主要用于解决多目标优化问题。NSGAIII 通过引入参
考点来保持种群的多样性和收敛性尤其适用于处理三个及以上目标的情况。在康养资源
优化配置问题中，需要同时考虑服务覆盖率、资源利用效率和成本这三个重要目标，
NSGA III 算法能够有效地处理这种多目标之间的权衡关系。它基于参考点的非支配排
序方法，可以生成均匀分布的Pareto 前沿，为决策者提供一系列具有代表性的最优解。

5.1 模型的优点
（1）贴合实际：本研究充分结合实际数据，考虑了康养资源分布的现状和居民健
康状况的差异，通过熵值法和多目标优化算法，构建了合理的模型。模型结果能够为北
京市康养城市建设提供科学依据。
（2）方法科学：模型运用了熵值法、TOPSIS 法、粒子群优化（PSO）、差分进化
算法（DE）和非支配排序遗传算法（NSGAIII）等多种方法，将复杂的资源优化问题转
化为可操作的数学模型，合理设置参数，确保模型的科学性和有效性。

度快、计算效率高等优点，能够有效解决多目标优化问题，适用于康养资源优化配置的
复杂场景。
（4）结果实用：模型得到的康养资源优化方案具有效率高、输出稳定、资源分配
均衡等特点，基本不存在资源浪费、服务不均衡等问题，在现有条件下能有效提高康养
服务的公平性和可达性。
5.2 模型的不足
（1）因素局限：实际应用中，政策变化、人口流动、经济发展等动态因素可能对
康养资源的合理配置产生重要影响，但本文未能充分考虑这些动态因素的影响，一定程
度上影响了模型的准确性和适应性。
（2）适用范围：本文提出的模型对于北京市的康养资源优化配置效果较好，但由
于未对其他城市或地区进行验证，模型的适用范围可能较窄。对于其他城市或地区，可
能需要根据当地实际情况进行调整。
（3）非线性因素：实际上，康养资源与居民健康之间的关系可能并非完全线性，
还可能受到多种非线性因素的影响，而本文将其作为线性因子处理，忽略了边际效应的
影响。

def calculate_gini(self, array):
"""计算Gini 系数，衡量资源分配的不平等程度"""
array = array.flatten()
if np.sum(array) == 0:
return 0
array = array / np.sum(array)
array = np.sort(array)
index = np.arange(1, len(array) + 1)
return np.sum((2 * index - len(array) - 1) * array) / (len(array) *
np.sum(array))
