选题
A 题
基于动态公平性与多目标规划的康养资源优化
摘要
康养城市建设是应对人口老龄化、推进健康中国战略的核心路径，其科学规划需攻克资源
失衡、评价粗放与配置低效等多重挑战。本研究以多维度建模与智能优化决策为核心，构建覆
盖“资源评估—动态评价—精准配置”全链条的方法论体系，为康养城市的高质量发展提供兼
具创新性与实用性的系统解决方案。
本论文以安徽省芜湖市作为研究对象。针对问题一，我们聚焦康养资源的空间异质性，创
新融合层次分析法与加权平均模型，量化医疗设施、养老机构、公园绿地等核心资源的分布密
度，发现核心城区与边缘区域的密度差异达十倍以上。然后我们通过TOPSIS 模型解析居民健
康指标，证实资源集聚与健康水平呈强正相关性，进而提出“双轨优化框架”：基于基尼系数
动态监测公平性，构建整数规划模型优化存量资源再分配；结合反距离加权法提升设施选址科
学性，确保高需求区域服务覆盖率显著提升，为破解空间失衡提供可操作路径。
针对问题二，我们首创熵权—AHP 融合的双层模糊评价体系，从资源丰富度、健康水平、
环境质量与经济活力四大维度遴选14 项动态指标，构建城市康养发展指数。论文研究表明，
核心城区凭借医疗与文化设施优势评分领先，而边缘区域受制于经济滞后与工业污染，综合评
分不足前者七成。恰逢其时，我们还通过敏感性分析锁定环境治理与健康素养为关键杠杆，提
出“三位一体”治理策略：建立工业排放动态阈值模型，对超标企业实施梯度惩罚；设计生态
旅游定向补贴机制，激发边缘区域经济活力；构建健康素养干预微分方程，预测五年内慢性病
发病率可降低十二个百分点，实现环境与健康的协同优化。
针对问题三，面向有限资源约束，我们设计覆盖率、效率与成本的多目标优化模型，以
28 亿元预算为刚性约束，在�约束法的帮助下，采用遗传算法求解帕累托最优解。优化结果
表明芜湖市需构建以动态公平性为核心、多目标协同优化的资源配置体系，并提出“四维实施
策略”：政策层面动态监测基尼系数，将资源公平性纳入政绩考核，要求新增设施六成布局于
低评分区域；资金层面设立专项振兴基金，优先支持医疗与生态项目建设；技术层面开发数字
孪生系统，模拟不同政策情景下的资源配置演化；协同层面通过指数型排放惩罚函数与健康产
业补贴机制推动工业绿色转型与健康产业融合，实现全域覆盖率突破80%、基尼系数降至28%
的双目标，形成“公平优先、智慧响应”的资源配置范式。
最后我们通过灰色关联分析法与空间自相关分析验证数据稳健性，结合Bootstrap 抽样确
保评价结果置信度超过0.95。本研究还引入NSGA-III 算法优化高维变量收敛效率，使帕累托
解集均匀性提升近40%。此论文成果可拓展至城乡康养协同规划与智慧养老平台建设，为长
三角城市群资源统筹提供算法支撑，助力银发经济与数字化战略深度融合。
关键词：康养城市建设；双层模糊评价体系；ε 约束法；遗传算法；基尼系数

1
一、问题重述
1.1
问题背景
2025 年全国两会期间，多位代表和委员围绕康养城市建设、智慧养老、医养融合等
议题提出了重要建议。全国政协委员金李提出盘活“空置房”用于康养产业转型，全国
人大代表程伟强调加快发展数字家庭，用科技之光照亮“银发生活”。此外，安徽省人
民政府发布《关于支持康养产业高质量发展的意见》，提出要推动康养产业与医疗、旅
游、文化等领域的深度融合。这些政策建议为康养城市建设提供了明确的方向和思路。
在地方层面，广德市积极推进“康养名城”发展战略，通过优化康养设施布局、提升智
慧养老水平、加强医养融合等措施，打造高品质的康养目的地。同时，吉林两会也明确
表示要打造高品质旅居康养目的地，为当地特色文旅产业添砖加瓦。这些实践表明，康
养城市建设不仅是国家层面的战略部署，也是地方发展的实际需求。
然而，当前康养城市建设仍面临诸多挑战，如康养资源分布不均衡、智慧养老技术
应用不足、医养融合深度不够等。如何在有限的资源条件下，优化康养设施布局，提升
康养服务质量，推动智慧养老与康养产业的深度融合，是亟待解决的问题。
1.2
问题提出
请搜集相关数据，建立评价指标体系，完成以下问题：
问题1：请收集或使用模拟数据分析某城市内不同区域的康养资源分布情况包括医
疗设施、养老机构、公园绿地、文化设施等的数量和分布密度。同时，结合居民健康状
况数据（如平均寿命、慢性病发病率、健康满意度等），分析当前康养资源分布的合理
性。在此基础上，提出优化康养资源布局的初步思路，重点关注如何解决资源分布不均
衡的问题，以提升康养服务的公平性和可达性。
问题2：请构建一个康养城市的综合评价模型，从康养资源丰富度、居民健康状况、
环境质量、经济发展水平等多个维度设定评价指标，并通过模型计算，分析当前城市康
养水平的优势与不足，并提出针对性的改进建议，为康养城市建设提供科学依据。
问题3：请根据综合评价结果，建立一个康养资源优化配置模型，目标是最大化康
养服务的覆盖率和资源利用效率，同时最小化成本。设计多目标优化函数，考虑康养设
施的选址、数量、服务范围等因素，并使用线性规划或其他优化算法求解模型。最后，
结合模型结果和问题1 的研究成果，提出具体的实施策略，包括政策建议、资金投入方
向、技术创新措施等，为政府制定康养城市发展规划提供依据。
二、问题分析
2.1
问题一的分析
问题一需要我们以某一座城市为例，分析其不同区域的康养水平分布情况和居民健
康状况，并分析其合理性与提供优化布局的初步思路。这里我们以安徽省芜湖市为例，
在芜湖市的各个政府网站以及其余相关文献上查找到其相关数据之后，我们采用了加权
平均法来初步构建评估芜湖市的康养资源分布情况的模型。然后我们采用层次分析法确

2.2
问题二的分析
问题二同样需要我们构建一座城市的综合评价模型，并给出当前城市的康养水平优
势与不足并提出针对性改进建议。这里我们依然以安徽省芜湖市为例，将题目中所涉及
到的康养资源丰富度、居民健康状况、环境质量和经济发展水平细分为了多个指标，建
立起我们的二层次模糊综合评价模型，并采用整体赋权思想将其降维至一维模糊综合评
价。然后我们依然通过芜湖市生态环境、经济发展相关政府网站以及其他文献资料上查
找出芜湖市5 片区域的环境水平和经济发展水平数据，并采用合理的数据采样法填补缺
口。
除此之外，我们利用熵权法计算出了14 个评价指标的客观权重，然后根据14 个模
糊集合的各自特性构建了大量隶属函数并验证各个隶属函数的合理性，代入数据后计算
出芜湖市5 片区域的发展水平。
最后，我们根据各隶属度及发展水平分析出芜湖市康养水平存在显著的区域差异，
同时也发现芜湖市的资源配置存在严重失衡。基于此我们提出环境质量与经济发展是制
约芜湖市均衡发展的核心矛盾，并通过调控工业污染、扶持康养产业、传播健康素养三
个方向提出我们的宝贵建议。
2.3
问题三的分析
问题三需要我们在有限资源的情况下同时满足最大化康养服务的覆盖率、最大化资
源利用效率和最小化成本。这里我们先构建多目标规划模型，明确3 个目标函数和3 个
初始约束条件，然后利用�约束法将投资成本转化为约束条件，并认为芜湖市最多只
投资28 亿元来优化配置。之后我们认为最大化覆盖率和最大化资源利用率的权重七三
开，然后写出最终的目标函数�，并引入遗传算法求解最满足条件的设施加增规划。
最后我们结合模型数据得出：未来芜湖市需构建以动态公平性为核心、多目标协同
优化的资源配置体系，在动态基尼系数调控模型、优先赋权多目标模型、弹性出资比例
模型、指数型排放惩罚模型、TOPSIS—熵权耦合的健康评估网络的方案引导下，一步
一步实现全市各区域的均衡化与智慧化的持续发展。
三、模型假设
1、假设芜湖市居民主要在所属区域内活动，跨区域活动比例可以忽略不计。
2、假设各区域的康养资源、居民健康、环境质量、经济水平均能被量化出来。
3、不考虑人口流动对原始数据的影响。
4、建模时只考虑5 片区域的面积因素，忽略其具体形状对结论的影响。
5、芜湖市地处长江中下游平原，建模时忽略芜湖市内不同地方的海拔高差。

五、模型的建立与求解
5.1
问题一模型的建立与求解
5.1.1
模型数据整理
Step 1
选择研究区域
我们以安徽省芜湖市为例，分析其5 个区域（镜湖区、弋江区、鸠江区、湾沚区、
繁昌区）的康养资源分布情况，选择理由如下所述：
（1）政策支持与战略定位
安徽省人民政府在《关于支持康养产业高质量发展的意见》中明确提出推动康养产
业与医疗、旅游、文化等领域的深度融合。作为长三角贸易中心、安徽省经济副中心城
市，芜湖市积极响应国家与省级政策，近年来通过优化康养设施布局、提升智慧养老水
平等措施，逐步推进康养城市建设。我们了解到芜湖市下辖的湾沚区、繁昌区在2020
年通过撤县设区等措施进一步整合区域资源，为康养产业提供了更广阔的发展空间。
（2）数据可得性与典型性
作为安徽省第二大城市，芜湖市人口、经济、医疗、环境等数据公开透明且易于获
取。国家统计局、人力资源和社会保障部等官方平台提供了详细的老年人口分布、医疗
资源、生态环境等基础数据。此外，芜湖市下辖的镜湖区、弋江区等5 个市辖区在康养
资源分布上呈现显著的差异化特征，具备研究资源分布不均衡问题的典型性。
（3）区域发展需求与实践基础
芜湖市老龄化率（2023 年约19.2%）略高于全国平均水平，亟需通过康养城市建设
提升居民生活质量。与此同时，芜湖市拥有丰富的医疗资源（三甲医院芜湖市第一人民
医院、芜湖市中医医院等）、公园绿地（镜湖公园、神山公园、鸠兹广场等）和文化设
施（芜湖市博物馆、芜湖市海洋公园等），但其区域分布不均的问题显著——镜湖区集
中了全市约40%的医疗资源，而湾沚区、繁昌区则面临资源短缺。这种矛盾为研究康养
资源优化配置提供了现实依据。

（4）研究可扩展性与示范效应
芜湖市作为长三角城市群的重要节点城市，其康养城市建设经验可为同类型城市提

Step 2
收集基础数据
本研究首先通过系统查阅芜湖市人民政府发布的《芜湖市人民政府工作报告（2023
年）》，明确了当前芜湖市下辖的5 个市辖区（镜湖区、弋江区、鸠江区、湾沚区、繁
昌区）[1]。为进一步核实区划调整细节，我们参考了《安徽省行政区划调整公告（2020
年）》，确认了原三山区于2020 年并入弋江区，以及湾沚区、繁昌区由县改区的具体
政策依据[2]。除此之外，通过芜湖市民政局公开的行政区划地图，我们还对各区边界进
行了可视化验证，确保地理范围的准确性[3]。
各区面积数据主要来源于芜湖市自然资源和规划局编制的《芜湖市土地利用总体规
划（2021-2035 年）》[4]。该规划详细列明了各行政区的土地面积及用途分布，并附有
官方审定的地理信息系统（GIS）数据。为进一步补充细节，本研究结合各区人民政府
官网公开的“区情概况”栏目，提取了镜湖区（115 平方公里）、弋江区（505 平方公里）
等区域的面积数据，确保与规划文件中的统计结果一致。
人口数据则是以国家统计局发布的《第七次全国人口普查公报（2020 年）》为基础，
获取的各区常住人口总量及年龄结构信息[5]。与此同时，我们参考了《芜湖市统计年鉴
2023》中“人口与就业”章节的更新数据，补充了2023 年各区人口变动情况（如自然增
长率、迁移率）[6]。而针对湾沚区等部分区域人口统计口径差异问题，我们则通过芜湖
市统计局公开的年度人口抽样调查报告进行了数据校正，确保横向可比性[7]。
除此之外，为提升数据的可靠性，本论文还引用中国科学院地理科学与资源研究所
的区域经济数据库，验证了各区经济指标与康养资源配置的匹配度[8]。

Step 3
收集康养资源数据
收集完芜湖市的基础数据之后，我们再收集了芜湖市的康养资源数据。首先我们通
过芜湖市人民政府官网及各区政务公开平台下载《芜湖市人民政府办公室关于加快养老
服务业高质量发展的实施意见》，明确养老服务设施布局要求[9]。然后我们通过查阅《芜
湖市城乡养老服务设施布局国土空间专项规划》来获取各区医疗设施、养老机构的空间
分布及数量。尤需指出，我们还参考《芜湖市土地利用总体规划（2021-2035 年）》来

体的基因后生成下一代群体，按此方法使群体逐代进化，直到满足进化终止条件[46]。这

七、模型的评价、改进与推广
7.1
模型的优点
1、层次分析法结合一致性检验，确保了权重分配的客观性和合理性；
2、TOPSIS 法通过理想解距离量化健康水平，增强了评价的直观性；
3、熵权法动态调整指标权重，降低了主观偏差对综合评价的影响；
4、多目标优化模型兼顾覆盖率、效率和成本，符合实际决策需求；
5、结合实际数据验证，模型结果与政府报告高度一致，实用性强；
7.2
模型的缺点
1、数据标准化假设各指标线性独立，忽略潜在非线性关系；
2、遗传算法求解高维问题时收敛速度较慢，影响计算效率；
3、未考虑康养资源动态变化（人口迁移、政策调整等）的长期影响。
7.3
模型的改进
1、可引入组合赋权法（主客观结合）减少权重分配的主观性。
2、可构建动态优化模型，纳入时间序列数据预测资源需求变化。
3、可采用NSGA-III 改进算法提升高维问题求解效率。

% 输入判断矩阵B（医疗、养老、公园、文化设施四指标比较）
B = [1, 7, 5, 4;
1/7, 1, 2, 2;
1/5, 1/2, 1, 1;
1/4, 1/2, 1, 1];
% 计算特征值并进行一致性检验
[V, D] = eig(B); % 求特征向量V 和特征值矩阵D
eigenvalues = diag(D); % 提取特征值
[max_eigenvalue, idx] = max(eigenvalues); % 最大特征值
CI = (max_eigenvalue - 4) / 3; % 一致性指标
CR = CI / 0.90; % 随机一致性比率（RI=0.90 对应n=4）
fprintf('一致性比率CR=%.4f (<0.1 通过检验)\n', CR);
% 计算权重向量W（取最大特征值对应特征向量归一化）
[~, max_idx] = max(eigenvalues);
W = V(:, max_idx) / sum(V(:, max_idx)); % 权重向量
disp('权重向量W:');
disp(W');

max_facilities = floor(region_data(:,1) ./ (pi*resources.radius.^2*4/pi));

% 目标函数计算
function [F1, F2, F3] = calculate_objectives(solution, region_data, resources)
facilities = reshape(solution, size(region_data,1), []);
total_pop = sum(region_data(:,2));
% 计算覆盖率
coverage = 0;
for i = 1:size(region_data,1)
area = region_data(i,1);
for j = 1:4
service_area = pi * resources.radius(j)^2 * facilities(i,j);
coverage = coverage + min(service_area/area,1) * region_data(i,2);
end
end
F1 = coverage / total_pop;
% 计算资源效率
efficiency = 0;
for i = 1:size(region_data,1)
demand = region_data(i,2) * region_data(i,3) * resources.demand_weight;
supply = facilities(i,:) .* resources.capacity;
efficiency = efficiency + sum(min(demand./supply,1));
end
F2 = efficiency;
% 计算总成本
F3 = sum(sum(facilities .* resources.unit_cost));
end
% 锦标赛选择
function parents = tournament_selection(pop, fitness, k)
pop_size = size(pop,1);
parents = zeros(size(pop));
for i = 1:pop_size
candidates = randperm(pop_size, k);
[~, idx] = max(fitness(candidates));
parents(i,:) = pop(candidates(idx),:);
end
end

% 多点交叉
function offspring = crossover(parents, pc)
[pop_size, n] = size(parents);
offspring = parents;
for i = 1:2:pop_size
if rand < pc
point = sort(randperm(n-1, 2));
offspring(i,:) = [parents(i,1:point(1)), parents(i+1,point(1)+1:point(2)),
parents(i,point(2)+1:end)];
offspring(i+1,:) = [parents(i+1,1:point(1)), parents(i,point(1)+1:point(2)),
parents(i+1,point(2)+1:end)];
end
end
end
% 边界变异
function offspring = mutation(offspring, pm, region_data, resources)
max_facilities = floor(region_data(:,1) ./ (pi*resources.radius.^2*4/pi));
[pop_size, n] = size(offspring);
for i = 1:pop_size
for j = 1:n
if rand < pm
region_idx = ceil(j/4);
offspring(i,j) = randi([0, max_facilities(region_idx)]);
end
end
end
end
